# Fibonacci Numbers
## 斐波那契数列
**Task.** Given an integer $n$,  find the last digit of the nth Fibonacci number $F_n$.

**Input Format.** The input consists of a single integer $n$.

**Constraints.** $0 ≤ n ≤ 45 $.

**Output Format.** Output $F_n$.
